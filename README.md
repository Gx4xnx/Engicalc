# 📐 EngiCalc 

EngiCalc is an open-source, community-driven web application designed to help engineering students master complex formulas by providing clear, step-by-step mathematical derivations and solutions rather than just a final numerical output.

## 💡 Features
* **Step-by-Step Breakdowns:** Every calculation is split into clear logical stages: identifying variables, stating formulas, substitution, and final calculation.
* **Completely Client-Side:** Lightweight and fast. Runs entirely in the browser using clean HTML, CSS, and vanilla JavaScript. No external dependencies or tracking.
* **Mobile Responsive:** Designed to work smoothly on laptops, tablets, and smartphones.

## 🤝 How to Contribute (Add Your Own Formulas!)
This project is designed to grow through community contributions. If you want to add a formula from your current engineering syllabus, follow these steps:

1. **Fork** this repository.
2. Open `index.html` and add a new menu item button in the `<aside>` sidebar, along with a corresponding user input container block.
3. Open `script.js` and write a clear calculation function that handles your math and pushes the step-by-step breakdown text to the DOM.
4. Open a **Pull Request** explaining the formula you added.
